# Панель приемной кампании 2026

## Как запустить

1. Откройте папку `pk-dashboard-2026`.
2. Дважды нажмите `index.html`.
3. Откройте сайт в Chrome или Edge.
4. Нажмите F11 для полноэкранного режима.
5. Для корректной работы локальных документов не переименовывайте папку `docs`.

## Как заменить файлы

Положите новые файлы в папку `docs` и сохраните имена:

- `Metodichka_PK_2026.pdf`
- `Podgotovka_dlya_kolledzhistov.pdf`
- `Programma_loyalnosti_2026_2027.pptx`
- `Brochure_bachelor.pdf`
- `Brochure_master.pdf`

Оригинальные сценарии консультаций лежат в папке:

- `docs/scripts/`

Карты дисциплин можно добавлять в:

- `docs/discipline_maps/`

## Как изменить тексты

Основные тексты, вопросы FAQ, направления, карточки материалов и ссылки находятся в файле:

- `assets/js/data.js`

## Что уже есть в первой версии

- главный экран под интерактивную панель;
- левое меню направлений;
- центральный hero-блок, который меняется при выборе направления;
- FAQ с модальными ответами;
- сетка материалов;
- модальные окна для скриптов, ДПО, скидок и отсутствующих файлов;
- локальные PDF открываются через страницу-просмотрщик;
- методические указания открываются как интерактивный справочник `method-guide.html`, PDF-оригинал доступен внутри него;
- DOCX-скрипты открываются как HTML-страницы внутри проекта;
- PDF-версии скриптов открываются из `docs/scripts_pdf/`;
- программа лояльности открывается как краткая и полная HTML-версия, а PDF-презентация лежит в `docs/Programma_loyalnosti_2026_2027.pdf`.
