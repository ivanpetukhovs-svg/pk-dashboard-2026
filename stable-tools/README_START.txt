Локальная stable-версия интерактивной панели

Как запустить:
1. Распакуйте папку pk-dashboard-2026-stable в удобное место.
2. Откройте файл start.bat.
3. Дождитесь открытия браузера.
4. Не закрывайте черное окно запуска, пока работает панель.

Что внутри:
- app - сама панель и все локальные материалы;
- tools - локальный сервер;
- start.bat - запуск одним кликом.

Важно:
- Не переименовывайте папки app и tools.
- Не перемещайте файлы внутри app.
- Внешние ссылки на сайт Московского Политеха и Google Drive требуют интернет.
- Локальные PDF, брошюры, методические материалы и скрипты работают без интернета.

Если страница не открылась:
- проверьте, что не закрыто окно start.bat;
- попробуйте открыть в браузере адрес http://127.0.0.1:8098/;
- если порт 8098 занят, обратитесь к ответственному за сборку.
